# User Guide

- Search books using chat
- Check availability
- Place holds
- Renew borrowed books
- Pay fines
- Register for library events
- Get personalized recommendations

- View chatbot analytics
- Monitor circulation stats
- Reduce manual queries
- Manage event registrations
