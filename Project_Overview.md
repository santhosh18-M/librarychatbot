# Project Overview

## Project Title
AI-Powered Library Services Chatbot with ScaleDown API Integration

## Problem Statement
Modern libraries handle thousands of books and patrons daily. Traditional library systems require manual assistance for catalog search, renewals, fine inquiries, and research help, leading to increased librarian workload and limited service hours.

## Solution
This project implements an intelligent chatbot that integrates with existing Library Management Systems (Koha/Sierra) and the ScaleDown API to provide 24/7 automated library services.

## Objectives
- Enable fast book search across 100,000+ records
- Allow patrons to place holds and renew books
- Provide personalized reading recommendations
- Compress catalog and summaries using ScaleDown API
- Reduce librarian queries by at least 50%

## Key Features
- Book catalog search
- Holds and renewals
- Reading recommendations
- Event registration
- Research assistance
- Fine payment support

## Technologies Used
- Backend: Node.js / Python (Flask)
- Frontend: React.js
- Database: PostgreSQL / MongoDB
- AI & NLP: ScaleDown API
- LMS Integration: Koha / Sierra REST APIs
- Version Control: Git & GitHub

## Target Users
- Library patrons
- Librarians
- Academic institutions
